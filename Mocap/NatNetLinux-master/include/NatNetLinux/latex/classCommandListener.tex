\subsection{\-Detailed \-Description}
\-Thread to listen for command responses. 

\begin{DoxyAuthor}{\-Author}
\-Philip \-G. \-Lee
\end{DoxyAuthor}
\-This class spawns a new thread to listen for command responses. \-This class is needed to retrieve the \hyperlink{classNatNet}{\-Nat\-Net} protocol version in use by the server.
